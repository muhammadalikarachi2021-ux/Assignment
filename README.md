# Student Attendance System

A professional assignment-ready attendance management system built with HTML, CSS, and JavaScript.

## Features

- Add student roll number and name
- Track present and absent status per student
- Save records in browser local storage
- View summary counts for present, absent, and total students
- Export attendance as a CSV file
- Clear all attendance records

## Project Structure

- `index.html` — main application markup
- `styles.css` — polished styling for a professional interface
- `script.js` — attendance logic and local storage management

## Usage

1. Open `index.html` in a browser.
2. Enter a student roll number and student name.
3. Click `Add Student`.
4. Mark each student as `Present` or `Absent`.
5. Click `Save Attendance` to persist the list.
6. Use `Export CSV` to download attendance records.

## Notes

- Attendance is stored locally in the browser and will remain available until cleared.
- This project is suitable for a school or college assignment presentation.
